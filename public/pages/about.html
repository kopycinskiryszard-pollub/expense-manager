O stronie

Expense Manager

Aplikacja do zarządzania finansami osobistymi, przygotowana jako projekt webowy z backendem API i modułowym frontendem.

Funkcjonalności

Po zalogowaniu użytkownik może prowadzić listę transakcji, planować budżety miesięczne, zarządzać celami oszczędnościowymi i przeglądać raporty finansowe.

Interfejs wspiera motywy kolorystyczne oraz tryb wysokiego kontrastu, a widoki są ładowane jako osobne części aplikacji.

Projekt

Frontend korzysta z prostego routingu po adresie hash, osobnych plików HTML dla podstron i modułów JavaScript ładowanych tylko tam, gdzie są potrzebne.

Backend udostępnia endpointy API dla uwierzytelniania, profilu, transakcji, budżetów, celów i raportów.
